# Journey Jenie - Travel Booking Platform

## 🌐 Live Project Link
🔗 **https://khushi-portfolio-18.netlify.app/*

## 📝 Project Overview
Journey Jenie is a clean, modern, and user-friendly web interface designed for a seamless travel discovery and holiday booking experience. The layout features smooth navigation patterns tailored to deliver a premium user experience.

## 🛠️ Tech Stack Implemented
- **Frontend Core:** HTML5, CSS3, JavaScript (ES6+)
- **Responsive Layout:** Flexbox & CSS Grid for universal cross-device adaptability (Mobile, Tablet, and Desktop).
- **Hosting Platform:** Netlify

## 🚀 Key Features
- Interactive landing sections showcasing popular global travel destinations.
- Fully responsive design optimized for all screen sizes.
- Clean, structured, and lightweight frontend codebase for faster loading speeds.
